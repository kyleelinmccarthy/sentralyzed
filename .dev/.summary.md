## What Is This?

Sentral is a collaborative workspace platform for teams. It provides project management, task tracking, communication (chat, forums), file sharing, real-time whiteboards, expense tracking, and more — all in one place.

## Stack Summary

- **Frontend**: Next.js 15 (App Router, React 19, Turbopack) + Tailwind CSS 4.0 + Zustand
- **Backend**: Hono.js REST API on Node.js + Drizzle ORM + PostgreSQL 16
- **Real-time**: WebSocket (ws) + Yjs CRDTs for collaborative editing
- **Cache/Sessions**: Redis 7
- **Validation**: Zod (shared between client and server)
- **Monorepo**: npm workspaces + Turborepo
- **Testing**: Vitest (unit) + Playwright (E2E)

## Architecture at a Glance

```
apps/api    → Hono.js REST + WebSocket (ports 3001, 3002)
apps/web    → Next.js 15 App Router (port 3000)
packages/shared → Types + Zod validators
packages/config → Shared tsconfig, eslint, prettier
```

**Key pattern**: Routes (`src/routes/`) → Services (`src/services/`) → Drizzle ORM → PostgreSQL
